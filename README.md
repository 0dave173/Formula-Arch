# Formula-Arch

A beige + light-blue Formula-1 inspired theme pack for Arch Linux
(Hyprland / Wayland).

## Features
- Hyprland config (auto-tiling + smooth animations)
- Waybar, Rofi, Kitty, Starship, Mako
- Custom GTK headerbar window buttons (SVG)
- Local-first AI wrapper ("RaceControl") via Ollama

## Install
```bash
cd formula-arch
./scripts/install.sh
